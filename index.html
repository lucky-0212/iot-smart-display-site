ESP32 • P10 LED • MQTT/HTTP

IoT‑Based Smart Dynamic Display Board & Notice Board System

Traditional notice boards are slow and tedious to update, often leaving outdated information on display. This project delivers a real‑time, remotely managed display board using an ESP32 and P10 LED matrices. Update messages instantly from a web or mobile app—no ladders, no paper, no delays.

Try the Demo Get Started

Key Features

  • Real‑time updates via Web/Mobile app (HTTP/MQTT/WebSocket)
  • Supports single or chained P10 32×16 LED matrix panels
  • Message scheduling, priorities, and auto‑expiry
  • Offline cache with graceful retry when internet drops
  • Admin dashboard with roles, audit log, and analytics
  • Secure updates: API keys, JWT, and device whitelisting

Why this solves the problem

Manual boards are inefficient and error‑prone. Our system centralizes content management and pushes updates in seconds. Time‑bound notices auto‑expire, emergency alerts preempt lower‑priority messages, and audit trails show who changed what—and when.
